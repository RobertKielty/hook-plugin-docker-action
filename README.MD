# Hook docker action

This action launches a hook instance with builtin plugins

## Inputs

## `plugin-to-run`

**Required** The name of the plugin to launch. Default `size`.

## `plugin-to-run`

**Required** The name of the plugin to launch. Default `size`.

## Outputs

none

## Example usage

uses: actions/hook@v1
with:
  plugin-to-run: 'blunderbuss'
